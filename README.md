# Ascii Generator

## Execution

Dans un terminal windows :

1. `compile.bat`
2. `run.bat`

Sinon :

1. `javac -cp lib/* @option.list @compile.list`
2. `java -Dfile.encoding=UTF-8 -cp bin;lib/* asciigenerator.App`
